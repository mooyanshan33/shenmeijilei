# Figma 画板规范（375–1440 / 6 列栅格）

## 栅格

- 画板宽度：`375 / 768 / 1024 / 1440`
- 列数：6
- 外边距：`16`（移动）→ `24`（平板）→ `32`（桌面）
- 栏间距：`16`

## Tokens

建议在 Figma 建立 4 个页面：

- `00 Tokens`：颜色/字体/圆角/阴影/动效
- `01 Components`：Card / Button / Input
- `02 Patterns`：列表、表单、详情页
- `03 Moodboard`：参考截图与旁注

本仓库对应的 tokens 文件位于 `docs/design-tokens.md`。
